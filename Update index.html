Hasil Pembelajaran HTML oleh Nunu

Aku mau nunjukin beberapa hal yang sudah aku pelajari: foto, list, bikin web, dan cara pasang foto di web.
Kita mulai dari penjelasan cara menambahkan gambar di web.

Cara Menambahkan Gambar ke Web

Untuk menampilkan gambar, kita pakai elemen <img> dan atribut: src (link gambar), alt (teks alternatif), dan width (ukuran gambar).
Contoh: 

<img src="https://cdn2.thecatapi.com/images/MTY3ODIyMw.jpg" alt="kucing hitam" width="300">

Cara Membuat List

Ada dua jenis list:

  • Ordered List (<ol>): list dengan nomor
  • Unordered List (<ul>): list dengan titik

Contoh Ordered List:

  1. Suka makan
  2. Suka mandi

Contoh Unordered List:

  • Suka makan
  • Suka mandi
  • Suka Shiva

Dasar Belajar Coding

Hal paling dasar dari coding adalah mengenali bahasa yang ingin dipelajari, contohnya HTML dan Python.

Beberapa Kode Dasar Python

Input: digunakan untuk menanyakan sesuatu ke pengguna. 

nama = input("Halo, boleh aku tahu siapa namamu?")

Print: digunakan untuk menampilkan jawaban atau respons. 

print("Halo " + nama + ", gimana kabarmu hari ini?")

If, Elif, Else: digunakan untuk memberikan respons berbeda tergantung jawaban. 

mood = input("Gimana kabarmu hari ini?")

if "senang" in mood or "bahagia" in mood: print("Wah senang dengarnya!") elif "sedih" in mood or "nangis" in mood: print("Aku juga sedih dengarnya, kalau mau cerita aku dengerin ya.") else: print("Wah dataku belum tahu itu, tapi aku tetap dengerin kok.")
